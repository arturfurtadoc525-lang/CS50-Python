# CS50's Introduction to Programming with Python 🐍

Este repositório contém as minhas resoluções para os problemas e projetos do curso **CS50P** de Harvard. O objetivo do curso é ensinar a linguagem Python de forma profunda, focando em boas práticas de programação, testes e automação.

## 🚀 Tecnologias Utilizadas
* **Python 3.12+**
* **Pytest** (Testes unitários)
* **Bibliotecas:** `sys`, `re`, `csv`, `PIL`, `requests`, entre outras.

## 🛠️ Destaques de Aprendizado

### Programação Orientada a Objetos (OOP)
Um dos maiores desafios foi a implementação da classe `Jar` (Semana 8), onde utilizei:
* **Encapsulamento:** Uso de atributos privados (`self._size`).
* **Properties:** Implementação de `@property` e setters para validação de dados.
* **Exceptions:** Levantamento de `ValueError` para garantir a integridade dos dados da classe.

### Testes Unitários
Aprendi a garantir a confiabilidade do código utilizando o framework `pytest`, criando suítes de teste que cobrem casos de borda e entradas inesperadas.
